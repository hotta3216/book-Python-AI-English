# Python×AIでつくる英会話練習アプリ

![表紙](images/Python%20x%20AI_Cover_03_fix_H1_small.png)

このリポジトリは、技術同人誌「Python×AIでつくる英会話練習アプリ」のソースコード・サンプルコード一式です。

本アプリは、GUIで条件を入力し、問題の読み上げ・録音・文字起こし・比較を行うことで英会話練習をサポートします。

## 主な機能

- 英文（質問・回答）の自動生成
- テキスト → 音声（TTS）
- 音声再生・録音・文字起こし（STT）
- ユーザー発話と正解文の比較（一致率算出）
- GUIによる操作

## 動作確認条件

- Python 3.12.4
- Windows11（macOS, Linuxにも対応していますが、動作確認環境はWindows11）
- ライブラリ（requirements.txtを参照）
  - FreeSimpleGUI==5.2.0
  - groq==0.31.0
  - numpy==2.1.1
  - openai==1.97.1
  - python-dotenv==1.0.0
  - sounddevice==0.5.2
  - soundfile==0.13.1

## ファイルについて

「english_app」フォルダ以下が、本書に記載しているソースコードです。

ファイル名に「OpenAI」とつくものは、8.1節「OpenAI APIの使⽤⽅法」でOpenAIのAPIを使用する際に書き換えたコードです。main.pyはGroq版のメイン処理コード、main_OpenAI.pyが、OpenAI版のメイン処理コードです。

「sample」フォルダは、2章に記載している動作確認用のサンプルコードです。

各章で実行している単体実行確認用コードを集めてコメントアウトしたものが、トップ階層にある「testrun.py」です。「testrun」は、各章で実行している単体実行確認用のコードです。「testrun」以下のファイルを単独で実行したい場合は、「english_app」直下にコピー・移動してから実行ください。
